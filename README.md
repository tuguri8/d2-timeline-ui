## D2-Timeline-ui

- D2 Campus Mini fest 7th 타임라인 만들기 프로젝트 프론트엔드
- 라이선스 : MIT
- 백엔드 저장소 : <https://github.com/tuguri8/d2-timeline-api>

## 개발환경

- React
- Spring Boot
- Apache Kafka
- H2 + Apache Cassandra

## Usage

```
yarn install
```

```
yarn start
```

## Used Open Source

- [React](https://github.com/facebook/react)

- [Material-UI](https://material-ui.com/)
